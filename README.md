# 🩺 Diabetes Prediction System

A Machine Learning project that predicts whether a person is likely to have diabetes based on medical input features.

---

## 📌 Project Overview

This project uses machine learning algorithms to analyze health-related parameters and predict the likelihood of diabetes. It is built as a practical implementation of data science concepts including data preprocessing, model training, and evaluation.

---

## 🎯 Objectives

- Predict diabetes using medical features
- Perform data preprocessing and feature analysis
- Train and evaluate machine learning models
- Build a reusable ML pipeline

---

## 📊 Dataset

- Dataset: Search PIMA Indians Diabetes Dataset(Kaggle)
- Features include:
  - Pregnancies
  - Glucose Level
  - Blood Pressure
  - Skin Thickness
  - Insulin
  - BMI
  - Diabetes Pedigree Function
  - Age
  - Placed here: data/diabetes.csv

## 🚀 Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/Diptesh-coder/Diabetes-Prediction.git
cd Diabetes-Prediction

2. Install dependencies: pip install -r requirements.txt

3. Run the project: python main.py

Train model: python src/train.py

## Run prediction:
python src/predict.py

## Run web app:
streamlit run app.py
